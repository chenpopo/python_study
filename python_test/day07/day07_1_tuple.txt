1、元组 - tuple
	1、什么是元组
		同list 一样，允许存放若干任意类型数据的容器
		但是元组是不可变的序列
	2、语法
		元组是使用()括起来，()内允许盛放若干元素，如果值放一个元素的话，元素后不要加","，用于区分对象还是元组
		1、使用字面值创建元组
			1、创建空元组
				t = ()
			2、创建非空元组
				t = (100, 200, 300)
				t = 100, 200, 300
				t = (100,)
				t = 100,
				
			元组的错误示例：
				1、t = (3000) t绑定的是3000
				2、x,y,z, = 100,200,300
					非元组赋值，而是将100赋值给x, 200赋值给y， 300赋值给z
				3、 x,y,z = (100, 200, 300)
					非元组赋值，效果同上
		2、使用构造函数创建元组
			1、创建一个空元组
				t = tuple()
			2、创建一个非空元组
				t = tuple(iterable) #使用可迭代对象生成一个元组
				示例：
					1、t = tuple()
					2、t = tuple(range(1,10,3))  -> (1,4,7)
					3、t = tuple("abc") -> ('a','b','c')
					4、t = tuple([1,2,3]) -> (1,2,3)
		
		3、元组的运算
			+, +=, ×, ×=
			< <= > >= == !=
			in / not in
			索引[]取值/切片[::] 取值
				索引和切片都不支持赋值(元组是不可变的)
		4、元组的方法
			>>> help(tuple)
			1、T.count(value)
			2、T.index(value)
			使用方式同列表
